Telegram Data Analyzer — это интеллектуальное веб-приложение для глубокого анализа экспортных данных Telegram. Проект позволяет превратить массив сырых сообщений в наглядный временной профиль пользователя и предсказать его возможное местоположение на основе упоминаний заведений (POI).
🚀 Основные возможности

    Анализ активности: Построение тепловых карт сообщений по часам и дням недели.

    NLP Обработка: Лемматизация и фильтрация текстов с использованием библиотеки Natasha и многопроцессорной обработки.

    NER (Named Entity Recognition): Автоматическое извлечение названий локаций (кафе, рестораны, бары, города) из переписки.

    Гео-аналитика:

        Геокодирование найденных названий через OpenStreetMap (Nominatim).

        Кластеризация точек методом DBSCAN для отсеивания шума.

        Предсказание текущего города проживания.

    Интерактивная карта: Визуализация найденных локаций на карте с помощью Folium.

    Экспорт: Возможность скачивания очищенных и лемматизированных данных в формате CSV.

🛠 Технологический стек

    Интерфейс: Streamlit

    Анализ данных: Pandas, NumPy

    NLP (Natural Language Processing): Natasha (Segmenter, MorphVocab, NER)

    Геоданные: Geopy, Folium, Scikit-learn (DBSCAN)

    Визуализация: Seaborn, Matplotlib

    Параллелизм: Multiprocessing (оптимизировано под Windows)

📂 Подготовка данных

Для анализа необходимо использовать стандартный экспорт из Telegram:

    Откройте Telegram Desktop.

    Перейдите в настройки чата или общие настройки -> Export Chat History / Export Telegram Data.

    Выберите формат JSON.

    Загрузите полученный файл в приложение.

⚠️ Производительность и советы

    Многопоточность: Приложение использует все доступные ядра CPU для лемматизации. При первом запуске нейросетевые модели могут подгружаться несколько секунд.

    Объем данных: Обработка 450,000 сообщений занимает около 1 часа (в зависимости от мощности CPU).

    Рекомендация: Для максимально быстрой работы загружайте историю конкретных чатов, а не весь архив аккаунта.

    Лимиты геокодера: При определении координат используется бесплатный сервис Nominatim. Он имеет ограничение на количество запросов, поэтому анализ большого списка заведений может занять время (1 запрос в секунду).
